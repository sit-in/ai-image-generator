AI图片生成器流程图（含淘宝充值）

graph TD A[用户] --> B[打开网页应用] B --> C[输入提示词] C --> D{点击生成按钮} D -->|是| E[检查用户积分] E --> F{积分是否足够?} F -->|是| G[显示加载状态] F -->|否| H[显示充值选项] H --> I[生成唯一校验码] I --> J[显示淘宝店铺信息] J --> K[用户淘宝下单] K --> L[用户输入校验码] L --> M{校验码是否正确?} M -->|是| N[自动充值积分] M -->|否| O[提示错误] N --> P[返回生成页面] G --> Q[调用 OpenAI API] Q --> R{API 调用是否成功?} R -->|是| S[显示生成的图片] R -->|否| T[显示错误信息] S --> U[提供下载选项] U --> V[用户下载图片] D -->|否| C

流程说明

  • 用户交互层
    • 用户打开网页
    • 在输入框中输入提示词
    • 点击生成按钮
  • 积分检查层
    • 调用 New API 检查用户积分
    • 积分不足时显示充值选项
    • 积分充足时继续生成流程
  • 充值流程
    • 系统生成唯一校验码
    • 显示淘宝店铺信息
    • 用户淘宝下单
    • 用户输入校验码
    • 系统验证校验码
    • 自动充值积分
  • 处理层
    • 系统显示加载状态
    • 调用 OpenAI API
    • 处理 API 响应
  • 结果层
    • 成功：显示生成的图片
    • 失败：显示错误信息
    • 提供图片下载功能
  • 错误处理
    • API 调用失败
    • 网络问题
    • 输入验证
    • 积分不足处理
    • 校验码错误处理
  • 状态管理
    • 加载状态
    • 错误状态
    • 成功状态
    • 积分状态
    • 充值状态
